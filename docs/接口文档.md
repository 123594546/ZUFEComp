# 接口文档（简版）

统一前缀：`/api`

## Auth
- `POST /auth/register` 学生注册
- `POST /auth/login` 登录（student/admin）
- `GET /auth/me` 当前用户信息（需登录）

## Activities
- `GET /activities` 活动列表（登录）
- `GET /activities/:id` 活动详情（登录）
- `POST /activities` 管理员新建
- `PUT /activities/:id` 管理员编辑
- `PATCH /activities/:id/status` 管理员改状态
- `DELETE /activities/:id` 管理员删除

## Enrollments
- `POST /activities/:id/enroll` 学生报名
- `DELETE /activities/:id/enroll` 学生取消
- `GET /me/enrollments` 学生我的报名

## Submissions
- `POST /activities/:id/submissions` 学生提交（multipart: file+note）
- `GET /me/submissions` 学生我的提交
- `GET /admin/submissions` 管理员审核列表
- `PATCH /admin/submissions/:id/review` 管理员审核

## Stats/Export
- `GET /admin/stats` 管理员统计
- `GET /admin/export/enrollments` 报名 CSV
- `GET /admin/export/submissions` 提交 CSV

## Files
- `GET /files/:submissionId` 按权限下载文件（admin 全部，student 仅自己）
