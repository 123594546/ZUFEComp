# 功能说明文档

## 项目背景
ZUFEComp 面向浙江财经大学学生，提供第二课堂活动与学科竞赛的一站式报名、材料提交、审核与统计管理流程，支持学生端与管理端协同，便于校内活动数字化管理。

## 功能清单
### 学生端
- 注册/登录/JWT 登录态持久化
- 活动列表（搜索、筛选、排序、分页、标签）
- AI 活动推荐（基于历史报名和提交记录）
- 活动详情（截止倒计时、剩余名额、报名/取消）
- 材料提交（文件上传+备注）
- 我的报名、我的提交、文件下载（权限控制）
- 积分中心（报名/提交/审核通过自动加分，支持明细查询）

### 管理端
- 活动管理（新增/编辑/删除/状态调整）
- 材料审核（通过/驳回+备注、批量审核）
- 智能审核建议（文件格式、大小、说明完整度评分）
- 数据看板（总活动/总报名/待审核、活动类型分布图）
- CSV 导出（报名导出、提交导出，支持状态筛选）

## 状态流转说明
- 活动状态：`draft -> published -> closed`
- 报名状态：`enrolled <-> cancelled`
- 材料审核状态：`pending -> approved/rejected`

## 扩展能力
- 多语言切换：学生端支持中文/英文导航快速切换。
- 智能推荐：`/api/me/recommendations` 根据历史行为打分推荐活动。
- 智能审核：材料上传时自动生成 AI 初审结果（pass/warn + reasons）。
