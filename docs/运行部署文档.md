# 运行部署文档

## 环境要求
- Node.js >= 20
- pnpm >= 9

## 一键启动
```bash
pnpm install
pnpm dev
```

## 默认端口
- Web: `5173`
- API: `3001`

## 配置说明
复制 `.env.example` 为 `.env` 可按需调整：
- `PORT`
- `JWT_SECRET`
- `VITE_API_BASE_URL`
- `DATA_FILE`
- `UPLOAD_DIR`
