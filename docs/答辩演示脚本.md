# 答辩演示脚本（3-5分钟）

1. **登录与角色分流（30s）**
   - 管理员与学生登录后自动进入不同端。
   - 亮点：JWT 鉴权 + 路由守卫 + 持久化。

2. **学生端流程（90s）**
   - 浏览活动列表（搜索/筛选/分页）
   - 进入详情查看倒计时、剩余名额并报名
   - 上传材料并在“我的提交”查看状态
   - 亮点：状态清晰、交互完整、错误提示友好。

3. **管理端流程（90s）**
   - 活动 CRUD 与状态切换
   - 审核材料（通过/驳回、备注、批量）
   - 亮点：管理闭环、权限隔离严格。

4. **数据与导出（40s）**
   - 看板查看总量、类型分布、通过率
   - 导出报名与提交 CSV
   - 亮点：支持离线评审与结果复用。

5. **工程与部署（20s）**
   - Monorepo、lowdb 本地持久化、上传权限下载接口。
